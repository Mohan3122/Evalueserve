# SharePoint Agent Creation and Sharing Guide

## Part 1: Create a SharePoint Agent using AI Actions

### Prerequisites
- Microsoft 365 Copilot license  
- Access to a SharePoint site or document library  
- Edit or Owner permissions  

### Step-by-step

#### 1. Open SharePoint Site or Document Library
- Go to your SharePoint site  
- Navigate to a document library, folder, or files  

#### 2. Select Content (Optional but Recommended)
- Select files or folders  
- This defines the knowledge scope for the agent  

#### 3. Click AI Actions
- In the command bar, select AI actions (or Create agent)  

#### 4. Choose Create Agent
- Select Create agent or Ask questions about this content  

#### 5. Configure the Agent

- Name  
- Description  
- Instructions:
```
Answer questions based only on selected documents.
Provide accurate summaries.
Include references where possible.
```
- Knowledge source (auto populated from selected content)

#### 6. Test the Agent
- Ask sample questions to validate responses  

#### 7. Save the Agent
- Click Create or Save

## Part 2: Share the Agent in Microsoft Teams

### Option 1: Share via Link

Steps:
1. Open the agent  
2. Click Share  
3. Copy the link  

#### Share in Teams Chat
- Open Teams → Chat  
- Paste and send the link  

#### Share in Teams Channel
- Post the link in a channel with context  

---

### Option 2: Use Agent via Copilot in Teams

Steps:
1. Open Teams  
2. Open Copilot or Chat  
3. Browse agents  
4. Select your agent  

#### Use in Chat or Channel
```
@Project Knowledge Agent summarize latest updates
```

---

### Option 3: Pin the Agent
- Open the agent in Copilot  
- Pin for quick access  

---

## Best Practices

- Scope agents to specific content  
- Use clear instructions  
- Use meaningful names  
- Share in relevant Teams channels
